แก้ไขข้อมูลสมาชิก
วิธีแก้ไขข้อมูลสมาชิก:
ไปที่แท็บ "ทำเนียบรุ่น" → กดที่การ์ดสมาชิก → กดปุ่ม "แก้ไขข้อมูล"

สิทธิ์การแก้ไข:
• Admin — แก้ไขได้ทุกคน + กำหนด role
• Editor ตร. — แก้ไขได้เฉพาะสมาชิก ตร.
• Editor ทบ. — แก้ไขได้เฉพาะสมาชิก ทบ.
• Editor ทอ. — แก้ไขได้เฉพาะสมาชิก ทอ.
• Editor ทร. — แก้ไขได้เฉพาะสมาชิก ทร.
• Member — แก้ไขได้เฉพาะข้อมูลตัวเอง (เบอร์/อีเมล/ไซร์เสื้อ/รูปภาพ)
ซิงค์ข้อมูล Sheet → Firebase
นำเข้าข้อมูลสมาชิกทั้งหมดจาก Google Sheet เข้า Firebase (collection: members)
หลังจากนำเข้าแล้ว แอปจะอ่านข้อมูลจาก Firebase เป็นหลัก ไม่ต้องพึ่ง Sheet อีก
* รูปภาพเก็บ URL ไว้ใน Firebase (ไฟล์ภาพจริงยังอยู่ใน Google Drive)
อัพเดท Sheet อัตโนมัติ (Google Apps Script)
วาง URL ของ Google Apps Script Web App ด้านล่าง เพื่อให้แก้ไขข้อมูลแล้วอัพเดทใน Sheet พร้อมกัน
ดูวิธีติดตั้ง Apps Script ▶
วิธีติดตั้ง / อัพเดท:
1. เปิด Google Sheet → Extensions → Apps Script
2. ลบโค้ดเดิมทั้งหมด แล้ววางโค้ดด้านล่างแทน
3. กด Deploy → New deployment (หรือ Manage → Edit → New version)
4. Execute as: Me / Who has access: Anyone
5. คัดลอก URL ใหม่ที่ได้มาวางในช่องด้านบน แล้วกด 💾 บันทึก
6. กด 🔗 ทดสอบ — ถ้าเห็น {"ok":true} แสดงว่าพร้อมใช้งาน

โค้ด Apps Script:
🔗 ผูกสมาชิกกับบัญชีนี้
